# File Storage

## الهدف
يوثق هذا المستند استراتيجية إدارة وتخزين الملفات في EduCore بما يضمن الأمان، وقابلية التوسع، وسهولة الوصول.

## أنواع الملفات
- صور الطلاب والموظفين.
- المرفقات التعليمية.
- المستندات الإدارية.
- التقارير المصدرة.
- ملفات النسخ الاحتياطي.

## خيارات التخزين
- Local Storage (للتطوير).
- Object Storage (S3 Compatible / Azure Blob) للإنتاج.

## المبادئ
- عدم تخزين الملفات الكبيرة داخل قاعدة البيانات.
- تخزين البيانات الوصفية (Metadata) فقط في قاعدة البيانات.
- استخدام أسماء ملفات فريدة (UUID).
- التحقق من نوع الملف وحجمه قبل الرفع.

## الأمان
- فحص الملفات عند الحاجة.
- تحديد أنواع الملفات المسموح بها.
- استخدام روابط مؤقتة للوصول إلى الملفات الخاصة.
- تشفير الملفات الحساسة أثناء النقل باستخدام HTTPS.

## قواعد العمل
- FS-0101: جميع الملفات ترتبط بسجل أعمال واضح.
- FS-0102: يمنع رفع الملفات التنفيذية أو غير الموثوقة.
- FS-0103: يدعم النظام حذف الملفات غير المستخدمة بشكل دوري.
- FS-0104: يجب تسجيل عمليات رفع وحذف الملفات الحساسة في Audit Log.
