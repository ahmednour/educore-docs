# System Context

## الهدف
يوضح هذا المستند حدود نظام EduCore، والجهات الخارجية التي يتكامل معها، وتدفق البيانات بين النظام والمستخدمين والخدمات الأخرى.

## الجهات المتفاعلة
- مدير النظام
- مدير الفرع
- المدرس
- الطالب
- ولي الأمر
- موظف الاستقبال
- المحاسب

## الأنظمة الخارجية
- مزود البريد الإلكتروني
- مزود الرسائل النصية (SMS)
- خدمة الإشعارات الفورية
- بوابة الدفع الإلكتروني
- مزود التخزين السحابي
- مزود الهوية (اختياري)

## تدفقات البيانات
- المستخدمون يتفاعلون مع بوابات الويب وتطبيقات الجوال.
- جميع الطلبات تمر عبر REST API.
- خدمات الخلفية تنفذ المهام المجدولة والإشعارات.
- قاعدة البيانات هي المصدر الرئيسي للبيانات.

## المبادئ
- SC-0101: يمنع وصول الأنظمة الخارجية مباشرة إلى قاعدة البيانات.
- SC-0102: جميع عمليات التكامل تتم عبر واجهات برمجة التطبيقات أو خدمات البنية التحتية.
- SC-0103: يجب تسجيل عمليات التكامل الحساسة في Audit Log.
- SC-0104: يدعم التصميم إضافة أنظمة خارجية جديدة دون تعديل جوهري في طبقة المجال.
