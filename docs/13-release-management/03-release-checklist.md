# Release Checklist

## الهدف
توفير قائمة تحقق موحدة للتأكد من جاهزية الإصدار قبل نشر EduCore على بيئة الإنتاج.

## قبل الإصدار
- اكتمال جميع الميزات المخطط لها.
- نجاح الاختبارات الآلية.
- اكتمال مراجعة الكود.
- تحديث التوثيق.
- تجهيز Release Notes.

## أثناء النشر
- اعتماد خطة النشر.
- التحقق من النسخ الاحتياطية.
- تفعيل المراقبة.
- تأكيد خطة التراجع.

## بعد النشر
- اكتمال اختبارات Smoke.
- التحقق من الخدمات الحرجة.
- مراقبة المؤشرات.
- إبلاغ أصحاب المصلحة.

## معايير النجاح
- اتباع كل إصدار إنتاجي لهذه القائمة.
- توثيق ومراجعة أي انحراف عنها.
