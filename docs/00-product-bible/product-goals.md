# أهداف المنتج

## الهدف الرئيسي
بناء منصة موحدة لإدارة جميع العمليات الأكاديمية والإدارية والمالية داخل المراكز التعليمية.

## أهداف الإصدار الأول

- إدارة الأعوام الدراسية.
- إدارة المراحل والصفوف والفصول.
- إدارة الطلاب وأولياء الأمور.
- إدارة المدرسين والمواد.
- تسجيل الطلاب.
- الحضور والغياب.
- إدارة الرسوم والمدفوعات.
- التقارير الأساسية.

## أهداف طويلة المدى

- دعم تعدد الفروع.
- دعم SaaS متعدد المستأجرين.
- تطبيقات الجوال.
- التكامل مع أنظمة الدفع.
- مساعد ذكي يعتمد على الذكاء الاصطناعي.
