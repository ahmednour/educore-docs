# قاموس المصطلحات

يُعد هذا المستند المرجع الرسمي للمصطلحات المستخدمة داخل النظام.

| المصطلح | التعريف |
|---------|---------|
| العام الدراسي | الفترة الزمنية التي تُدار خلالها الدراسة داخل المركز. |
| المرحلة | مجموعة تعليمية مثل الروضة أو الابتدائية أو الإعدادية. |
| الصف | المستوى الدراسي داخل المرحلة. |
| الفصل | مجموعة الطلاب داخل الصف. |
| البرنامج | مسار الدراسة مثل عربي أو لغات. |
| المادة | المادة الدراسية التي تُسند إلى مدرس. |
| الطالب | شخص مسجل في المركز التعليمي. |
| ولي الأمر | المسؤول عن الطالب ماليًا وإداريًا. |
| المدرس | المستخدم المسؤول عن تدريس مادة أو أكثر. |
| التسجيل | ربط الطالب بعام دراسي وصف وفصل. |
| الرسوم | المبالغ المستحقة على الطالب. |
| الدفعة | عملية سداد مالية مقابل الرسوم. |
| الحضور | تسجيل حالة حضور أو غياب الطالب. |
| الاختبار | تقييم أكاديمي للطالب. |
