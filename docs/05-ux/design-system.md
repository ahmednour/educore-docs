# Design System

## الهدف
يوثق هذا المستند نظام التصميم (Design System) في EduCore لضمان الاتساق وإعادة استخدام المكونات عبر جميع التطبيقات.

## المبادئ
- مصدر موحد للمكونات.
- تصميم Mobile First.
- دعم RTL وLTR.
- الالتزام بإمكانية الوصول.

## المكونات الأساسية
- Buttons
- Inputs
- Cards
- Tables
- Dialogs
- Navigation
- Forms
- Badges
- Alerts
- Toasts

## التصميم المرئي
- نظام ألوان موحد.
- Typography متسقة.
- Spacing باستخدام Design Tokens.
- أيقونات موحدة.

## قواعد العمل
- DS-0101: يمنع إنشاء مكونات مكررة دون مبرر.
- DS-0102: جميع المكونات تدعم الوضعين الفاتح والداكن.
- DS-0103: جميع المكونات تدعم RTL.
- DS-0104: يجب توثيق أي مكون جديد قبل استخدامه.
