# Error States

## الهدف
يوثق هذا المستند إرشادات عرض حالات الخطأ في EduCore لضمان تقديم رسائل واضحة تساعد المستخدم على فهم المشكلة واتخاذ الإجراء المناسب.

## المبادئ
- استخدام لغة بسيطة ومباشرة.
- توضيح سبب المشكلة عندما يكون ذلك مناسبًا.
- اقتراح الخطوة التالية أو إعادة المحاولة.
- تجنب عرض التفاصيل التقنية للمستخدم النهائي.

## أمثلة
- انقطاع الاتصال.
- انتهاء الجلسة.
- عدم توفر الصلاحية.
- حدوث خطأ غير متوقع.

## قواعد العمل
- ERR-0101: تعرض جميع الأخطاء رسالة مفهومة.
- ERR-0102: توفر إمكانية إعادة المحاولة عندما يكون ذلك مناسبًا.
- ERR-0103: تسجل التفاصيل التقنية في نظام المراقبة فقط.
- ERR-0104: تستخدم جميع حالات الخطأ مكونات موحدة من Design System.
