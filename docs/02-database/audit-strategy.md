# Audit Strategy

## الهدف
تحديد استراتيجية موحدة لتتبع جميع العمليات الحساسة داخل قاعدة بيانات EduCore بما يضمن الامتثال، وسهولة التحقيق، والحفاظ على سلامة البيانات.

## المبادئ
- تسجيل جميع العمليات الحساسة تلقائيًا.
- عدم السماح بتعديل سجلات التدقيق.
- ربط كل عملية بالمستخدم والوقت والكيان المتأثر.
- دعم البحث والتصفية والتصدير.

## البيانات المسجلة
- UserId
- EntityName
- EntityId
- Action
- OldValues
- NewValues
- IpAddress
- UserAgent
- CreatedAt

## قواعد العمل
- AU-0101: جميع عمليات الإنشاء والتعديل والحذف المنطقي تُسجل.
- AU-0102: العمليات المالية وتغيير الصلاحيات تُسجل دائمًا.
- AU-0103: يمنع حذف سجلات التدقيق من التطبيق.
- AU-0104: الوصول إلى السجل يتطلب صلاحية خاصة.

## متطلبات التنفيذ
- دعم Audit Interceptors.
- دعم Correlation ID لتتبع الطلبات.
- إمكانية أرشفة السجلات القديمة دون فقدانها.
