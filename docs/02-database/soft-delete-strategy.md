# Soft Delete Strategy

## الهدف
تحديد آلية موحدة للحذف المنطقي داخل EduCore لضمان الاحتفاظ بالبيانات التاريخية وإمكانية الاسترجاع ومنع فقدان البيانات التشغيلية.

## المبادئ
- لا يتم حذف السجلات التشغيلية حذفًا نهائيًا.
- تستخدم الحقول DeletedAt وDeletedBy وIsDeleted.
- جميع الاستعلامات تستبعد السجلات المحذوفة افتراضيًا.
- يمكن استعادة السجلات المحذوفة حسب الصلاحيات.

## قواعد العمل
- SD-0101: يمنع الحذف النهائي للطلاب والمدرسين والتسجيلات والفواتير.
- SD-0102: الحذف النهائي مسموح فقط لبيانات النظام المؤقتة أو عبر أدوات الصيانة.
- SD-0103: جميع عمليات الحذف المنطقي تُسجل في Audit Log.
- SD-0104: لا يجوز إنشاء سجل جديد بنفس المفتاح المنطقي إذا كان هناك سجل محذوف منطقيًا يسبب تعارضًا.

## متطلبات التنفيذ
- دعم Global Query Filters.
- دعم Restore.
- دعم Purge للبيانات المؤرشفة وفق سياسات المؤسسة.
