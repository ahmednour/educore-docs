# الكيانات الأساسية (Core Entities)

## الهدف
يوثق هذا المستند الكيانات الأساسية في قاعدة بيانات EduCore ومسؤولية كل كيان والعلاقات المنطقية بينه وبين بقية النظام.

## الكيانات

### Branch
يمثل الفرع الذي تُدار بداخله جميع العمليات.

### AcademicYear
يمثل العام الدراسي النشط أو المؤرشف.

### Program
يمثل البرنامج الدراسي (عربي / لغات).

### Stage
يمثل المرحلة التعليمية (KG، ابتدائي، إعدادي...).

### Grade
يمثل الصف الدراسي داخل المرحلة.

### Classroom
يمثل الفصل الدراسي الفعلي.

### Subject
يمثل المادة الدراسية.

### Teacher
يمثل بيانات المدرس وإمكانيات التدريس.

### Student
يمثل بيانات الطالب الأساسية.

### Parent
يمثل ولي الأمر والعلاقة مع الطلاب.

### Enrollment
يمثل تسجيل الطالب في عام دراسي وفصل محدد.

## ملاحظات
- جميع الكيانات تستخدم UUID كمفتاح أساسي.
- جميع الكيانات تدعم Soft Delete.
- جميع الكيانات تحتوي على حقول الإنشاء والتعديل والتدقيق.
- سيتم توثيق كل كيان بالتفصيل في ملفات منفصلة لاحقًا.
